# mmk-cc-workshop

Claude Code 워크샵 실습을 위한 스타터 프로젝트입니다.

## 시작하기

### 1. Fork

이 저장소를 본인 GitHub 계정으로 Fork합니다.

### 2. Claude Code 웹에서 열기

[claude.ai/code](https://claude.ai/code) 에서 Fork한 저장소를 연결합니다.

### 3. 코딩 시작

세션이 시작되면 자동으로 환경 체크리스트가 실행됩니다:
